# Support

## Before opening an issue

Please check the following:

- [ ] You are using **Redmine 6.x** or higher
- [ ] You are using **Ruby 3.x** or higher  
- [ ] You have run `bundle exec rake redmine:plugins:migrate RAILS_ENV=production`
- [ ] You have **fully restarted** Redmine after installing the plugin
- [ ] You have checked the [README](README.md) troubleshooting section
- [ ] You have searched [existing issues](../../issues)

## Reporting a bug

Use the **Bug report** issue template and include:
- Your Redmine, Ruby and Rails versions
- The relevant lines from your error log (`docker logs` or `production.log`)
- Steps to reproduce the problem

## This plugin is community-maintained

This plugin is provided **as-is**, maintained on a best-effort basis.
Response times may vary. Pull requests with fixes are very welcome!

## Commercial support

If you need guaranteed support or custom development, please contact the maintainer directly.
